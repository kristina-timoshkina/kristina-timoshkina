## Мои проекты

### Telegram-бот для поиска вакансий

Бот помогает собирать, хранить и показывать подходящие вакансии. В проекте реализованы пользовательский режим, административные команды, база данных, ежедневная подборка и AI-анализ вакансий.

### Telegram-бот для информационного проекта

Бот помогает пользователям получать структурированную информацию через понятное меню, а администратору — управлять содержимым проекта.
